# File Tab Power (Acode)

Plugin para Acode com ações rápidas de gerenciamento de arquivos/pastas diretamente pela paleta de comandos.

## Recursos

- Criar arquivo
- Criar pasta
- Renomear
- Copiar
- Mover
- Excluir

## Instalação rápida

1. Compacte a pasta `acode-file-tab-power` em `.zip`.
2. No Acode: **Plugins > Install plugin > Local plugin**.
3. Selecione o `.zip`.
4. Reabra o app e use a paleta de comandos para procurar por `File Tab Power`.

## Observações

- O plugin usa `fsOperation` quando disponível.
- Em versões mais antigas do Acode, métodos podem variar (o plugin já tenta fallbacks).
- Você pode adaptar os comandos para abrir menus contextuais em árvore de arquivos, se quiser ir além.
